## 谷歌加速器 Windows 版

谷歌加速器 Windows 客户端专为 PC 用户优化，支持 Windows 10 和 Windows 11 系统。

### 系统要求

- 操作系统：Windows 10（1903+）/ Windows 11
- 处理器：Intel Core i3 或同等性能以上
- 内存：4GB RAM 以上
- 网络：宽带连接

### 下载安装步骤

1. 访问谷歌加速器官网下载 Windows 客户端
2. 双击 `GoogleJiasuqi-Setup.exe` 安装包
3. 如弹出安全警告，点击"更多信息"→"仍要运行"
4. 按照安装向导完成安装
5. 安装完成后桌面会自动生成快捷方式

### Windows 版特色功能

- **开机自启** — 可设置开机自动启动并连接
- **系统托盘** — 最小化到托盘，不占任务栏空间
- **智能分流** — 仅加速 Google 相关流量，不影响国内网站速度
- **快捷键** — 支持自定义快捷键快速连接/断开

### 常见问题

**Q: Windows 防火墙阻止连接？**  
A: 首次运行时允许谷歌加速器通过防火墙即可。

**Q: 杀毒软件报毒？**  
A: 谷歌加速器安全无毒，请添加信任或临时关闭杀毒软件安装。

**Q: 连接后某些国内网站变慢？**  
A: 开启智能分流模式，仅加速 Google 服务，国内网站走本地网络。
