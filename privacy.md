    <h1 class="page-title">隐私政策</h1>

    <div class="post-content" style="max-width:760px;margin:0 auto;">
      <p>最后更新日期：2026年6月</p>

      <p>谷歌加速器（以下简称"我们"）高度重视用户的隐私保护。本隐私政策说明了我们如何收集、使用和保护您的个人信息。</p>

      <h2>一、信息收集</h2>
      <p>我们仅收集运行服务所必需的最少信息：</p>
      <ul>
        <li><strong>账户信息</strong> — 注册邮箱地址、加密后的登录凭证</li>
        <li><strong>支付信息</strong> — 支付由第三方支付平台处理，我们不存储您的银行卡信息</li>
        <li><strong>设备信息</strong> — 操作系统版本、客户端版本号（用于兼容性优化）</li>
      </ul>

      <h2>二、我们不收集的信息</h2>
      <ul>
        <li>❌ 浏览历史</li>
        <li>❌ 流量内容</li>
        <li>❌ DNS 查询记录</li>
        <li>❌ 连接日志</li>
        <li>❌ IP 地址（连接时不记录）</li>
      </ul>

      <h2>三、信息使用</h2>
      <p>收集的信息仅用于：</p>
      <ul>
        <li>提供和维护加速服务</li>
        <li>处理支付和订单</li>
        <li>发送服务相关通知</li>
        <li>改进产品体验</li>
      </ul>

      <h2>四、信息安全</h2>
      <p>我们采用行业标准的加密技术和安全措施保护您的数据，包括端到端加密传输和安全存储。</p>

      <h2>五、联系我们</h2>
      <p>如对本隐私政策有任何疑问，请联系：<a href="mailto:support@googlejiasuqi.com">support@googlejiasuqi.com</a></p>
